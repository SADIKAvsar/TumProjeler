❌ **Frozen** (Action Needed):

- No loading indicators
- Static error message

- Black/blank screen

## Response Format

Respond ONLY with valid JSON:

```json

  "freeze_type": "loading" | "popup" | "stuck" | "disconnect" | "crashed",
  "recommended_action": "wait" | "close_popup" | "teleport" | "reconnect" | "restart",

```

**Field Explanations:**

- `is_frozen`: Is this actually a problem requiring intervention?
- `freeze_type`: Category of issue
- `recommended_action`: What the bot should do next
- `confidence`: How certain you are (0.7+ needed for restart action)
- `visual_evidence`: Describe specific visual clues
- `wait_time_seconds`: If waiting, how long before rechecking?
